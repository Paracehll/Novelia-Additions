# Novelia 相關腳本與工具集

本倉庫包含一組用於優化 [Novelia](https://n.novelia.cc/) 使用體驗的用戶腳本（Userscripts）及相關工具。

## 腳本與工具說明

### 1. Novelia 論壇搜索增強 (`novelia_forum_search.js`)
*   **功能**：為論壇的「小說交流」版塊新增搜索功能。
*   **特色**：
    *   **本地快取**：自動掃描並快取帖子，搜索反應極快。
    *   **進階過濾**：支持關鍵字、作者 (`a:"作者名"`)、起始時間 (`f:"YYYYMMDD"`) 及結束時間 (`t:"YYYYMMDD"`)。
    *   **自動補全**：提供作者名自動補全與語法提示。
    *   **UI 優化**：支持面板折疊，搜索條隨文字輸入變色提醒。
*   **注意事項**：首次使用或清除瀏覽器快取後需進行完整掃描。

### 2. 哈梅林爬取器 v2.0 (`hameln_spider.v2.js`)
*   **功能**：爬取 [syosetu.org (ハメルーン)](https://syosetu.org/) 的小說內容。
*   **特色**：
    *   **多格式匯出**：支持將小說合併為 TXT 或 EPUB 下載。
    *   **範圍爬取**：可指定爬取特定章節範圍。
    *   **自動同步**：支持一鍵將爬取的內容加入 Novelia 本地書架（Sakura 空間）。
*   **注意事項**：請遵守目標網站的訪問限制，避免過快爬取導致封鎖。

### 3. EPUB 合併工具 (`epub_merge.html`)
*   **功能**：純前端的 EPUB 檔案合併工具。
*   **特色**：
    *   **隱私安全**：所有處理均在瀏覽器本地完成，不需上傳檔案至服務器。
    *   **自動整理**：自動提取元數據並重新編排章節序號。
*   **使用方式**：直接用瀏覽器開啟此 HTML 檔案，將多個 EPUB 拖入即可合併。

### 4. Novelia 小說鏈結分享按鈕 (`novelia_novel_share_clipboard_btn.js`)
*   **功能**：在小說列表與文庫頁面添加「複製」按鈕。
*   **特色**：
    *   **Markdown 格式**：點擊後自動複製為 `[標題](連結)` 格式，方便在 Discord 或論壇分享。
    *   **批量複製**：支持多選累加複製。
    *   **快捷鍵**：支持 `Ctrl+Q` 清空、`Ctrl+V` 查看、`Shift+R` 刷新。

### 5. Novelia Forum-Edit 頁尾固定 (`novelia_thread_footer.js`)
*   **功能**：優化論壇編輯頁面的操作體驗。
*   **特色**：
    *   **固定 UI**：將標籤導航欄與提交按鈕固定在頁面底部，方便長文編輯時隨時切換或提交。
    *   **快速捲動**：新增「回頂」與「回底」按鈕。

## 安裝方式
1.  安裝瀏覽器擴充功能：[Tampermonkey](https://www.tampermonkey.net/)。
2.  點擊本倉庫中對應的 `.js` 檔案，選擇「Raw」視圖，Tampermonkey 通常會自動彈出安裝界面。
3.  對於 `epub_merge.html`，下載後直接用瀏覽器開啟即可使用。
